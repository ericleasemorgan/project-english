Besides search, Project English includes a number of extra tools supporting additional functionality:

  • List assertions - Given a lemmatized noun and a lemmatized verb, this tool will output a list positive or negative assertions in the form of noun-verb-(noun or adjective). It is useful for extracting the definitions of things, listing what things have, or listing what things do.
  • Compare & contrast - Given a conjunction (and, or, not, or but) as well as a part-of-speech (noun, verb, or adjective) this tool will list the frequency of conjunctive phrases: this and that, one or another, etc. This tool is useful for answering the questions, "What & how are compared & contrasted in this text or corpus?"
  • Search document - Given a Project English identifier and a word, this tool returns a list of lines from the given text containing the word -- a "keyword-in-context" search result.
  • Describe - Given a lemmatized noun, this toolreturns a frequency list of two-word adjective phrases, thus "describing" the input. This is useful for gauging the sentiment towards a thing.
  • Plot dispersion - Given a Project English identifier and a list of words, visualize where the words occur in a text.
  • Make word cloud - Given an Project English identifier and a type of word, this tool returns a word cloud illustrating frequencies.
  • List ngrams - Given a Project English identifier and an integer, this form returns a frequency list of ngrams (i.e. one-word, two-word, three-word phrases) from the identified text.
  • List noun phrases - Given a Project English identifier, return a frequency list of subject and object parts of a sentence.
  • Create "study carrell" - Given a set of one more more Project English identifiers, this page will create a sub-collection of the whole.
  • View as table - Given a set of one more more Project English identifiers, this tool will return a tabled view of the set. This enables you to sort the search results in any number of ways.
  • Get bibliographics - Given a set of one or more identifiers, this program will return bibliographic metadata describing each item.
  • List URLs - Given a list of Project English identifiers, this page will return a list of corresponding URLs pointing to plain text versions of the content. The resulting list can be fed to an Internet spider so the content can be cached locally.
  • Download search results - Given a set of one more more Project English identifiers, this tool creates an archive of the reader's search results. This will enable the reader to "read" the results in any number of ways.
  • Compute log-likelihood - Given two Project English identifiers, this tool returns the "G2 score" for all the words in first document of the given identifiers. Words with higher G2 scores are more significant for the first document when compared to the second.
  • List parts-of-speech & named-entities - Use this page to list the frequency of selected parts-of-speech and named entities existing in a document. This is is useful for gauging what & how things are discussed.
